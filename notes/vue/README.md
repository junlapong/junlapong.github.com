# Vuejs

https://vuejs.org

<img src="https://vuejs.org/images/logo.png" width="200">

## Prerequisite

- [Nodejs](https://nodejs.org)
- Yarn

```
	npm install -g yarn
```

## Installation

### Vue CLI

```
	npm install -g @vue/cli
```

## New Project

```
vue create my-app
# then follow step to setup project

cd my-app
yarn serve
```
