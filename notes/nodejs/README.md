## Installation

- use [brew](https://brew.sh/)

```
brew install node
```

- use [nvm](https://github.com/nvm-sh/nvm)

```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.3/install.sh | bash
```
